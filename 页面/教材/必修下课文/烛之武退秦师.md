# 《烛之武退秦师》／《左传》

## 课程思政素材

### 文本类

1. 经典选读《春秋左传注》中华书局1990年版。
2. 阅读材料《左传·僖公三十年》。
3. 拓展阅读《东莱左传博议》《秦晋崤之战》。
4. 联系阅读《曹刿论战》。
