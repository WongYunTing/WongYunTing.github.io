# 《哈姆莱特》／莎士比亚

## 课程思政素材

### 文本类

1. 选自《莎士比亚全集》人民文学出版社。
2. 联系阅读《李尔王》《麦克白》《奥赛罗》。
3. 对比阅读《赵氏孤儿》。
4. 比较阅读《窦娥冤》《雷雨》。
5. 拓展阅读茅盾《莎士比亚与〈哈孟雷特〉》。
